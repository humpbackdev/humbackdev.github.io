What is Humpback?

Humpback is a Docker-based environment for Drupal development that makes it easy to build Drupal 8 sites allowing you to focus only in building your sites and not your environment. It has several tools useful for any developer involved in the process.

It's composed of 2 projects Humpback and Humpback Generator. The former is the development environment and the latest is the generator to create projects using that development environment.

Why Humpback?

Docker is a whale, so, we wanted to be a whale too. While researching for whales, we loved the idea of naming our project as Humpback because that's one of the species most seen at Costa Rica

Getting started

Humpback generator is extremely easy to use. Just install it from npm project page and generate a project in a new folder:

          npm install -g yo
          npm install -g generator-humpback
And then: 
          mkdir my-project && cd my-project
          yo humpback

In the generated project, you'll have the necessary documentation to continue your journey. You also can access that documentation in USAGE.md
